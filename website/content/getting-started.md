# 快速开始

本指南将帮助你在 5 分钟内完成 JAcoworks 的安装和首次对话。

## 系统要求

| 平台 | 最低版本 | 建议配置 |
|------|---------|---------|
| macOS | 12 Monterey 及以上 | Apple Silicon (M1+), 8GB 内存 |
| Windows | 10 (1809) 及以上 | 64 位, 8GB 内存 |
| Linux | Ubuntu 22.04 / Fedora 38 及以上 | x86_64 或 aarch64, 8GB 内存 |

> **注意**: JAcoworks 的 AI 运算在云端完成，本地不需要 GPU。网络连接是必须的。

## 下载安装

### macOS

1. 前往 [下载页面](https://jaco.jingao.club/download) 下载 `.dmg` 安装包
2. 打开 `.dmg` 文件，将 JAcoworks 拖入「应用程序」文件夹
3. 首次打开如遇提示，请参考 [下载页面](https://jaco.jingao.club/download) 的安装说明

### Windows

1. 前往 [下载页面](https://jaco.jingao.club/download) 下载 `.exe` 安装包
2. 双击运行安装程序，按向导完成安装
3. 安装完成后，从开始菜单或桌面快捷方式启动

### Linux

> Linux 版本即将发布，敬请期待。

## 首次启动

### 获取激活码

JAcoworks 目前支持两类账号：

- **激活码账号**：通过激活码注册后使用用户名/密码登录
- **京奥内部账号**：通过飞书登录（无需激活码）

如需注册激活码账号，请先获取激活码：

- 联系团队管理员

### 注册账号（激活码）

1. 启动 JAcoworks，进入登录界面
2. 点击「激活码注册」
3. 输入你的激活码、用户名和密码
4. 点击「注册」完成账号创建

### 京奥内部登录（无需激活码）

1. 启动 JAcoworks，进入登录界面
2. 点击「飞书登录」
3. 在飞书授权页面确认授权并返回应用

### 登录

支持两种登录方式：

- **用户名 + 密码**: 适用于激活码注册账号
- **飞书 SSO**: 适用于京奥内部账号（仅京奥内部可用）

## 创建第一个对话

1. 登录后进入主界面，左侧为会话列表，右侧为对话区域
2. 点击左侧边栏顶部的 **「+ 新对话」** 按钮
3. 在底部输入框中输入你的问题，按 `Enter` 发送
4. AI 会实时显示回答

**示例问题**:

```
帮我写一个 Python 脚本，读取 CSV 文件并生成数据摘要
```

## 选择 AI 模型

JAcoworks 支持多种主流大语言模型，你可以根据任务需要切换：

1. 点击顶部工具栏的 **模型选择器**
2. 从下拉列表中选择合适的模型：
   - **Claude Sonnet 4.6** — 日常对话首选，平衡速度与质量
   - **Claude Opus 4.6** — 复杂任务，深度推理
   - **GPT-5.3 Codex** — 编程任务优选
   - **Gemini 3.1 Pro** — 长文本和多步推理
   - **GLM-5** — 中文写作优选

> 详细的模型对比和选型建议，请参阅 [AI 模型选择指南](guide/models)。

## 设置工作空间

工作空间让 AI 助手能够直接访问你的项目文件：

1. 点击顶部工具栏的 **文件夹图标**
2. 在系统文件选择器中选择你的项目目录
3. 确认后，AI 助手即可读写该目录下的文件

**示例**: 设置好工作空间后，你可以这样提问：

```
阅读当前目录下的 README.md，帮我补充安装说明部分
```

> 工作空间的详细使用方法，请参阅 [工作空间功能](guide/workspace)。
