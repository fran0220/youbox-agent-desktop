# 更新日志

## v1.9.0 (2026-04-09)

- 🏗 **架构升级**：外部能力统一迁移到 Skill + CLI 架构，搜索、资产生成、浏览器操作均通过独立 CLI 工具执行，响应更快、扩展更灵活
- 🔍 **全新搜索引擎**：集成 ai-search 多源聚合搜索，替代旧版单源搜索，结果覆盖面和质量大幅提升
- 🎨 **资产创作增强**：图片、视频、音频、音乐、语音、3D 模型等全类型资产生成统一通过 asset-gateway 完成，支持更多格式和编辑模式
- 📊 **演示文稿升级**：slide-deck 技能全面重构，SVG → PPTX 转换管线优化，Python 运行时预装依赖，开箱即用
- 🌐 **浏览器自动化**：新增 agent-browser 工具，支持网页数据抓取、表单填写、截图等操作
- 🪟 **Windows 体验优化**：所有运行时（Python / bash / Node.js）和 CLI 工具均随安装包捆绑，无需用户额外安装任何依赖
- 🐛 开发模式技能热更新修复，删除源文件后立即生效，无需手动清理缓存
