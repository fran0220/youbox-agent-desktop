# 定时任务

定时任务让你可以安排 AI 助手在指定时间自动执行工作——即使你的电脑已经关机。

## 什么是定时任务

你可以用自然语言告诉 AI 助手「每天早上 9 点帮我整理昨天的工作日志」或「20 分钟后提醒我开会」，AI 助手会自动创建定时任务，在云端按时执行。

**定时任务的特点：**

- 在云端运行，不依赖你的本地电脑
- 支持一次性、固定间隔和自定义时间表
- 执行结果可通过飞书推送通知
- 失败自动重试

## 支持的调度方式

### 一次性任务

在指定时间执行一次，执行后自动删除。

```
20 分钟后提醒我提交周报
明天上午 10 点帮我生成项目进度汇总
```

### 固定间隔任务

按固定时间间隔重复执行。

```
每 2 小时检查一次邮箱，帮我整理未读邮件摘要
每 30 分钟统计一次数据变化
```

### 自定义时间表

使用 cron 表达式定义更灵活的执行计划。

```
每个工作日早上 9 点帮我生成今日待办清单
每周五下午 5 点帮我写本周工作总结
```

## 如何创建定时任务

### 方法一：自然语言（推荐）

在对话中直接告诉 AI 助手你的定时需求，它会自动理解并创建：

```
帮我设一个定时任务：每天早上 9 点总结昨天的工作内容
```

AI 助手会确认任务详情后创建定时任务。

### 方法二：任务面板

1. 在顶部工具栏打开 **任务面板**
2. 点击 **「新建定时任务」**
3. 填写任务描述和执行时间
4. 确认创建

## 管理定时任务

你可以在对话中随时管理已有的定时任务：

```
列出我所有的定时任务
删除「每日工作总结」这个定时任务
暂停所有定时任务
```

也可以在任务面板中查看任务列表、执行历史和手动触发任务。

## 执行结果通知

定时任务执行完成后，结果会通过以下方式通知你：

- **任务面板**: 查看执行历史和详细结果
- **飞书通知**: 京奥内部用户可收到飞书消息推送

## 使用建议

- **描述越具体越好**: 明确告诉 AI 助手你期望的输出格式和内容
- **先测试再定时**: 创建定时任务前，先在对话中手动执行一次确认效果
- **合理设置频率**: 避免过于频繁的定时任务
- **一次性任务优先**: 临时需求用一次性任务更灵活

## 注意事项

- 定时任务在云端执行，需要网络连接才能创建和管理
- 任务执行时使用你当前选择的 AI 模型
- 失败的任务会自动重试，重试间隔逐步增大
- 一次性任务执行后默认自动删除
