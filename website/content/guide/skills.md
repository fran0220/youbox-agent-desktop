# 技能系统

技能（Skills）可以理解为 AI 助手的"专业模式"。

当你处理特定任务（如数据分析、合同审阅、海报生成）时，开启对应技能会明显更稳、更快。

## 怎么使用技能

你可以用两种方式：

1. **自动触发**：直接说需求，AI 助手会自动选合适技能
2. **手动指定**：在消息前加 `/技能ID`

示例：

```text
/data-analysis 分析 sales.csv，给我一份按月份的趋势结论
```

```text
/poster 帮我做一张新品发布海报，风格简洁高级
```

## 每个内置技能怎么用

### `web-search`

- **什么时候用**：你需要"最新""官方""外部"的信息
- **输入建议**：问题 + 时间范围 + 来源偏好（如官网/媒体）
- **示例**：`/web-search 帮我找 Notion AI 最近 3 个月更新`

### `content-extract`

- **什么时候用**：你已经有 URL，想快速读取正文
- **输入建议**：给完整链接，并说明你要"摘要/翻译/提取数据"
- **示例**：`/content-extract 读取这个链接并给出 5 条关键结论：https://...`

### `document-processing`

- **什么时候用**：要处理 Word、Excel、PDF、CSV 文件
- **输入建议**：明确文件名 + 想要的输出格式
- **示例**：`/document-processing 把 report.docx 改成一页摘要并导出 docx`

### `data-analysis`

- **什么时候用**：要做统计、分组、趋势、可视化
- **输入建议**：给文件 + 分析维度 + 输出形式（表/图/结论）
- **示例**：`/data-analysis 分析 orders.csv，按周输出销量趋势和异常点`

### `finance`

- **什么时候用**：预算对比、财务报表整理、偏差分析
- **输入建议**：说明周期（按月/季度）+ 指标 + 关注阈值
- **示例**：`/finance 对比 Q1 预算与实际，偏差超 10% 的条目单独说明`

> 说明：财务技能用于分析草稿，正式报表请由财务人员复核。

### `legal`

- **什么时候用**：合同、NDA、条款风险初审
- **输入建议**：说明你方角色（甲方/乙方）+ 希望重点关注条款
- **示例**：`/legal 审查 service-agreement.docx，重点看赔偿和违约责任`

> 说明：法律技能用于初筛与草稿，不构成法律意见。

### `marketing`

- **什么时候用**：写营销文案、活动方案、品牌语气统一
- **输入建议**：目标人群 + 场景 + 语气 + 渠道（邮件/公众号/社媒）
- **示例**：`/marketing 给我 5 条小红书标题，风格真实不夸张`

### `feishu`

- **什么时候用**：你要在飞书内处理消息、日历、审批、通讯录或多维表格
- **输入建议**：明确目标动作 + 对象 + 时间范围（如"给谁发消息""查哪天日程"）
- **示例**：`/feishu 给产品组发消息：今天 17:30 前提交周报，并抄送我`

> 说明：`feishu` 技能仅支持京奥内部账号使用。

### `poster`

- **什么时候用**：活动宣传、新品发布、品牌海报
- **输入建议**：主题 + 受众 + 风格 + 比例（竖版/方图）
- **示例**：`/poster 新品发布会海报，竖版，偏高级科技风`

### `infographic`

- **什么时候用**：把长内容压缩成一张看得懂的图
- **输入建议**：给原文或要点 + 目标读者 + 展示重点
- **示例**：`/infographic 把这份行业报告做成一张"高密度信息图"`

### `slide-deck`

- **什么时候用**：周会、路演、项目汇报
- **输入建议**：汇报对象 + 页数 + 风格 + 目标（说服/汇报）
- **示例**：`/slide-deck 基于这份 PRD 做 12 页管理层汇报`

### `xhs-images`

- **什么时候用**：做小红书图文卡片、知识干货连图
- **输入建议**：主题 + 人群 + 想要的语气（种草/科普/经验）
- **示例**：`/xhs-images 把这篇"护肤入门"整理成 8 张图文`

### `nano-banana-pro`

- **什么时候用**：从文案生成图片，或对现有图片做改图
- **输入建议**：主体 + 风格 + 构图 + 清晰度（1K/2K/4K）
- **示例**：`/nano-banana-pro 生成一张米白背景的产品静物图，4K`

### `video-gen`

- **什么时候用**：需要生成短视频或动态视觉内容
- **输入建议**：描述画面主题 + 风格 + 时长
- **示例**：`/video-gen 生成一段 5 秒的产品展示动画，科技感`

### `game-dev-ai`

- **什么时候用**：想快速制作一个可运行的小游戏
- **输入建议**：游戏类型 + 玩法描述 + 风格偏好
- **示例**：`/game-dev-ai 做一个太空射击小游戏，像素风`

### `building-skills`

- **什么时候用**：你想把重复工作封装成"自己的技能"
- **输入建议**：说明技能名称、用途、触发词
- **示例**：`/building-skills 创建"客户周报生成器"，输入销售表自动输出周报`

## 实用建议

- **先给结果目标**：例如"我要一页总结""我要 6 张图文"
- **再给限制条件**：例如字数、风格、受众、截止时间
- **技能不生效时**：在消息前手动加 `/技能ID`，成功率最高
- **文件类任务**：先确认文件已在当前工作空间中
