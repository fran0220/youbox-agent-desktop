# 协作模式

协作模式是 JAcoworks 的云端协作功能。你可以把它理解成"给你单独准备的一台云端工作电脑"，用来处理需要协作或需要与本机分离的任务。

## 什么是协作模式

在协作模式下，AI 助手在你的专属云端工作区运行。

和标准模式相比，区别可以简单理解为：

| 特性 | 本地模式 | 协作模式 |
|------|---------|-------------|
| 工作位置 | 你的电脑本地 | 云端独立工作区 |
| 文件位置 | 本地工作目录 | 云端工作目录 |
| 适合场景 | 日常高频使用 | 协作和云端连续作业 |
| 对本机影响 | 直接使用本机环境 | 对本机环境影响更小 |

## 开通云端工作区

### 首次使用

1. 点击顶部工具栏的 **云端按钮**
2. 系统检测到你还没有云端工作区，会自动分配
3. 等待系统准备环境（通常 30-60 秒）
4. 连接成功后，自动打开云端对话面板

### 工作区状态

你的云端工作区可能看到以下状态：

| 状态 | 说明 |
|------|------|
| 运行中 | 正在运行，可以正常使用 |
| 已停止 | 已停止，需要启动 |
| 已暂停 | 长时间未使用，再次使用时会自动恢复 |
| 创建中 | 正在创建中 |
| 异常 | 创建或启动失败 |

长时间不使用的云端工作区会被自动暂停以节省资源，再次使用时会自动恢复。

## 云端对话

云端协作的对话体验与本地模式基本一致：

1. 在云端面板的输入框中输入消息
2. 消息发送到你的云端工作区
3. 云端 AI 助手处理请求并实时返回结果
4. 对话实时显示在界面中

云端连接支持自动重连和心跳保活，网络短暂波动不会丢失对话内容。

## 什么时候建议用协作模式

### 推荐场景

- 你希望把任务放到云端，和本机项目隔离
- 需要多人协作同一类任务
- 你会在多台设备之间切换使用

### 不推荐场景

- 你要频繁直接修改本机项目文件
- 你当前网络不稳定
- 任务只需要快速改几处本地内容

## 与本地模式如何配合

推荐策略：

1. **先在本地模式起草**：需求整理、文件初稿、快速修改
2. **再切协作模式做协作**：跨设备接续、团队评审、云端统一处理
3. **最后回本地收尾**：落盘、归档、提交

### 注意事项

- 协作模式中的文件默认在云端，不会自动同步到本地
- 网络质量会直接影响使用体验
- 连接异常时会自动重连，如持续断开可先切回本地模式继续工作
- 每个用户对应一个云端工作区
- 长时间不使用的工作区会自动暂停，再次使用时自动恢复
