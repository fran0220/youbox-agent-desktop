# Poster Sizes

## Standard Poster Sizes

| Name | Aspect Ratio | Dimensions (px) | Use Case |
|------|-------------|-----------------|----------|
| Movie Poster | 2:3 | 1080×1620 | 电影院海报、关键视觉 |
| Event Poster | 11:17 | 1100×1700 | 演唱会传单、活动宣传 |
| Square | 1:1 | 1080×1080 | 社交媒体、专辑封面 |
| Horizontal | 16:9 | 1920×1080 | 数字屏幕、横幅 |
| Story/Mobile | 9:16 | 1080×1920 | 手机壁纸、Story |
| Portrait | 3:4 | 1080×1440 | 通用竖版 |
| A4 Print | ~1:1.414 | 2480×3508 | 打印海报 |
| A3 Print | ~1:1.414 | 3508×4961 | 大幅打印 |

## Resolution Tiers

| Tier | Image Size | Use Case |
|------|-----------|----------|
| 1K | ~1024px long edge | 草稿、快速验证 |
| 2K | ~2048px long edge | 社交媒体、屏幕展示 |
| 4K | ~4096px long edge | 打印、高清展示 |

## Safe Zones

For print posters, keep critical content within safe margins:
- **Bleed area**: 3-5mm outside trim
- **Safe zone**: Minimum 10mm inside trim for critical text
- **Title zone**: Top 30-40% of poster
- **Action zone**: Bottom 10-20% for CTA/details
