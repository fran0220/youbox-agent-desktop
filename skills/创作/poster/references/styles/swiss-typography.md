# swiss-typography

Clean international typographic style with mathematical grid precision.

## Color Palette

- Primary: Pure white (#FFFFFF) or off-white (#F8F8F8)
- Secondary: Pure black (#000000) for text
- Accents: One bold color — signal red (#FF0000), Swiss blue (#0057B8), or vibrant yellow (#FFD700)
- Minimal color use — impact through typography, not color

## Visual Elements

- Strict grid system (visible or implied)
- Asymmetric but balanced layouts
- Large amounts of whitespace
- Geometric shapes as design elements
- Photography used documentarily, not decoratively
- Clean horizontal and vertical lines
- Mathematical precision in spacing

## Typography

- Helvetica, Akzidenz-Grotesk, or similar grotesque sans-serif
- Extreme size contrast between heading and body text
- Flush-left, ragged-right alignment
- Bold weight for titles, light/regular for details
- Typography AS the design element — type is the hero

## Best For

Corporate, technology, premium brands, modern events, editorial, architecture, any context requiring authority and clarity
