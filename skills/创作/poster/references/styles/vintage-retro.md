# vintage-retro

Nostalgic warmth with aged textures and classic design sensibility.

## Color Palette

- Primary: Warm cream (#F5E6CC), dusty rose (#D4A0A0), sage green (#9CAF88)
- Secondary: Burnt orange (#CC5500), mustard yellow (#E8A317), faded teal (#5F9EA0)
- Accents: Muted red (#C44536), chocolate brown (#3E2723)
- Overall: Desaturated, warm-shifted, as if sun-faded

## Visual Elements

- Aged paper and parchment textures
- Distressed and worn edges
- Halftone dot patterns
- Visible print registration marks
- Rounded corners and soft shapes
- Vintage illustration style (hand-drawn feel)
- Stamp and badge motifs
- Retro borders and ornamental frames

## Typography

- Vintage serif fonts (Clarendon, Rockwell, slab serif)
- Script and hand-lettering for accents
- Letterpress texture on type
- Arched and curved text layouts
- Inline and shadow type treatments
- Distressed type edges

## Best For

Travel posters, cafe/restaurant, craft products, music (jazz/blues/country), heritage brands, nostalgic events, farmers markets
