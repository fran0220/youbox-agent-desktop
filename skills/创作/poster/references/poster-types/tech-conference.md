# tech-conference

Tech conference poster — clean, modern, and information-dense.

## Purpose
Developer summits, tech conferences, hackathons, webinars, and product launch events.

## Composition
- **Hero zone** (40%): Abstract tech visual or speaker portrait
- **Info zone** (40%): Clean geometric layout with structured information
- **Action zone** (20%): Registration details and sponsor logos

## Visual Elements
- Abstract geometry and circuit patterns
- Code snippets as decorative elements
- Gradient backgrounds with depth
- Network nodes and connection lines
- Data visualization elements as texture

## Text Placement
- **Title**: Conference name — bold and modern
- **Subtitle**: Keynote speakers and theme
- **Body**: Date, venue, and agenda highlights
- **CTA**: Registration info and early-bird pricing
- **Footer**: Sponsor logos row, hashtag

## Recommended Pairings
- `brutalist`: developer culture and raw energy
- `cyberpunk-neon`: cutting-edge and futuristic
- `technical-schematic`: engineering precision
- `swiss-typography`: corporate and professional
